NAME

archive - selectively archives and deletes old files on a remote host

SYNOPSIS

/n.B archive/n [/n.B -c/n.I <config>/n] [/n.B -d/n] [/n.B -l/n] [/n.B -v/n] [/n.B -z/n] /n.I <host_name>/n |/n.B -V/n

AVAILABILITY

archive is a Bourne shell script which should work equally well on all versions of UNIX and Linux.

DESCRIPTION

archive creates a full backup of local or remote systems using /n.I tar/n or /n.I lftp's mirror/n command.

OPTIONS

-c <config_file>
The default /n.I configuration file/n is /n.B /usr/local/etc/archive.cf/n. An alternative configuration file may be set with the /n.B -c/n option on the command line.

-d sets the script to debug mode (ie. runs with the shell option -x)

-l turns on logging using the syslog facility. This is particularly useful where statistics are being compiled for presentation to the client, see report.

-v Sets the script to verbose mode. Each significant action will be reported on standard out.

-z Use /n.I gzip/n to compress the tar file and append /n.I .gz/n to the archive filename. The /n.B -z/n option is meaningless when a remote system is being archived as the /n.I tar/n file is always compressed prior to transmission across the network.

-V prints version details to standard out and exits.

EXAMPLES

5 0 * * * su fulford -c "/usr/local/etc/archive -l albert"

The above is an extract from a root crontab file and illustrates the typical use of the script in practice. All the local file systems on the host "albert" will be backed up to device (or file) nominated in the default configuration file /n.I /usr/local/etc/archive.cf/n. Initiation and completion of the script will be logged using the /n.I syslog/n facility.

Note that the script is executed with an ordinary user id. This is to facilitate remote access where root logins are denied. Sudo is used within the script where root access is required.

.BR . The hostname albert is being resolved locally and does not require a fully qualified domain.

BUGS

The script is quite crude and has little or no error checking.

FILES

archive.cf.
